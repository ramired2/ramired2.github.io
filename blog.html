Challenges I faced

Rendering the page for the second task was the most complicated and difficult for me. At first, I wasn't able to get the posts to show up at all. I was still confused about the method that I was supposed to be using. The fact that the json data didn't have a name for the array confused me. I was testing some things and found that using [] resulted in the page being rendered, however, three additional empty posts kept showing up. After learning that using 'this' instead of using [] worked as well I played around with it and found 'this.price' looped through it correctly without the three extra posts.
